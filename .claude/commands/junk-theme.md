# junk juice テーマ・タイトル設計

junk_juiceのNote記事テーマを評価し、有料・無料の判断・ペルソナ設定・タイトル決定まで行うシステム。

使い方：`/junk-theme [テーマ案]`（テーマなしで起動した場合はStep 0の質問から開始）

---

## アカウント概要

- **アカウント**：junk juice（https://note.com/junk_juice）
- **ジャンル**：60代推し活エッセイ（Juice=Juice × シニアライフスタイル）
- **キャラクター**：60代女性、ユーモア・自虐あり、ポジティブ、「動けるうちにやりたいことを全力でやる」
- **記事の型**：感動・高揚型体験記7割・やらかし系3割
- **バズ実績**：article_02「1日3公演で沼落ちした話」いいね155がベンチマーク
- **必須タグ**：#創作大賞2026 #エッセイ部門 #60代 #推し活
- **ターゲット**：ハロプロ・J=Jファン20〜40代（主軸）＋「先延ばしにしている何かがある30〜50代」（マネタイズ層）
- **Phase1（現在）**：フォロワー100人未満 → 無料記事専念。Phase2（100人〜）で有料200円〜解禁

## Step 0：テーマなしの場合

「最近の推し活で印象に残った体験・エピソードを教えてください（感動・やらかし・初めての経験など何でも）」と1つだけ質問する。

---

## Step 0.5：有料・無料の判断

**この判断をせずに先に進まない。**

- **種別指定あり**：ユーザーが「有料」「無料」を明示した場合はその指定に従う
- **種別未指定・フォロワー100人未満**：無料記事を推奨する
- **有料記事の条件（共通）**：「他では読めない希少性」があり、読者が払う価値を感じるベネフィットが明確な場合のみ

```
【判定】：無料記事 / 有料記事
理由：（1文で）

＜有料の場合のみ追記＞
有料にできる根拠：
想定価格：（200円・300円・500円）
無料パートで見せるもの：
有料パートに残すもの：
```

## Step 1：ペルソナ設定

このテーマを読んで最も心を動かされる「たった一人」を設定する。汎用的な属性ではなく、具体的な一人の人物として描くこと。

出力する項目：
- 仮名・年齢・属性・現在の状況
- 今この人が抱えている気持ち（感情レベルで）
- この記事に出会う場面（いつ・どこで・どんな気持ちで読むか）
- この記事を読んで得たいもの・変えたいもの

「このペルソナで進めますか？」と確認する。

---

## Step 2：テーマのシビア評価

以下の3名が独立した視点で審査する。忖度・曖昧な合格判定は禁止。

**【感情担当】** バズ記事（いいね155）と同等以上の「感情の山」があるか。「読んでよかった」「シェアしたい」「コメントしたい」と思わせるか。

**【逆算担当】** junk_juiceの長期資産になるか。フォロワー増加・有料記事への導線につながるか。1回読まれて終わりの消耗品になっていないか。

**【一次情報担当】** 実体験に基づき「この人でないと書けない」具体的なシーン・エピソードが存在するか。一般論・ありきたりな話になっていないか。

```
【感情担当の評価】：
【逆算担当の評価】：
【一次情報担当の評価】：

【判定】：採用 / 却下

＜却下の場合＞
却下理由：（ロジカルに・忖度なしで）
代替テーマ案：（1案・理由付き）

＜採用の場合＞
このテーマがペルソナに刺さる理由：（2〜3文）
記事の型：感動・高揚型 / やらかし系
```

却下の場合はStep 2を繰り返す（代替案をユーザーが承認したら次へ）。

## Step 3：SEOキーワード確認（タイトル前に必ず実行）

① ペルソナが「夜にスマホで検索しそうな言葉」を3〜5語句リストアップ
② WebSearchで実在確認・競合記事の切り口を把握
③ タイトル用キーワード1〜3語を選ぶ（複合キーワード可）
④ H2見出し組み込み用の関連ワード2〜3語をピックアップ

採用基準：実際に検索されている・タイトルに入れて不自然でない・体験感情系を優先（「おすすめ」「ランキング」より）

確認結果の出力（必須・引き継ぎ情報に含める）：

```
【SEOキーワード確認結果】
確認したサジェスト候補：
競合記事の主な切り口：
差別化余地（junk_juiceならでは）：
採用キーワード（タイトル用）：
H2見出しへの組み込み候補ワード：
```

## Step 4：タイトル設計

採用テーマとSEOキーワードをもとにタイトル案を3つ生成する。

**ケープルズの4特徴（優先順位順）**
1. **得になる**（最重要）：読者がこの記事で何を得るか明確か
2. **好奇心**：「どういうこと？」と続きを読みたくなるか
3. **新情報**：「この人ならでは」と感じさせる要素があるか
4. **手っ取り早さ**：「自分にも近い話だ」と感じさせるか

**固有ルール**
- SEOキーワードを自然に組み込む
- 60代・シニア目線（無理に若い言葉・流行語を使わない）
- ユーモア・自虐の一言が差別化になる
- 否定的・暗いイメージを避ける（やらかし系も前向きなトーン）
- 具体的な数字・場面があると信頼性が上がる（「3公演」「1時間」等）
- 45字以内

```
案A：[タイトル]
案B：[タイトル]
案C：[タイトル]

| 評価軸 | 案A | 案B | 案C |
|---|---|---|---|
| 得になる要素 | | | |
| 感情フック | | | |
| SEOキーワード | | | |
| 差別化（この人ならでは） | | | |
| 60代らしさ | | | |
```

「どの案を選びますか？または修正指示をください」と確認する。
