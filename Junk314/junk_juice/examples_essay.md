## 冒頭（読者を1行目で掴む）

**❌ Bad（整いすぎたAI文・60代の声が消えている）**
```
近年、シニア世代の間で「推し活」が注目されています。
60歳の私がアイドルのコンサートに参加した体験をお伝えします。
```
**⭕ Good（実記事）**
```
ねえ、みんな「推し活」ってしてる？

巷でよく耳にするし、若い子たちがアツくなってるイメージが強いアレ。
まさか60歳になった私が、その当事者になるなんて夢にも思っていなかった。
```
**何が違うか**: Badは社会論から入っている（writing_core 5大ルール①違反）。Goodは**読者への語りかけ**で始まり、「アレ」という口語の距離感と「まさか私が」の自虐予告で続きを読む理由を作る。

## 本音の見せ方（きれいごとにしない）

**⭕ Good（実記事）**
```
私の第一反応は、ぶっちゃけこう。
「いやいや、女性アイドルのコンサートって、そもそもおっさんしかいないでしょ？ しかも、私みたいなおばさんが見たところで、何が楽しいのよ……」
```
```
最初の感想は、正直「うわ、イタイわ〜……」だった。
```
**注釈**: 偏見や失礼な第一印象を**先に自分の口から言い切る**のがこのアカウントの生命線。あとの「沼落ち」との落差がバズの構造そのもの（偏見→転換→熱狂の一発逆転ストーリー）。AIはここを「様々な方がいらっしゃいました」と丸めてしまう。丸めた瞬間に60代の実在感が死ぬ。

## 転換点（感情が動く瞬間は「音」と「発見」で書く）

**❌ Bad（因果を説明してしまう）**
```
同世代のファンが多いことに気づき、親近感が湧いて楽しめるようになりました。
```
**⭕ Good（実記事）**
```
「待って。この人たち……私と同世代じゃん」

そう、現場にいるオタクたちの多くが、白髪混じりだったり、お腹が出ていたり、私と同じ「シニア世代」だったのだ。
なんだ、若者の文化に紛れ込んじゃったと思って縮こまっていたけど、ここにいるの同級生みたいなもんだわ。
そう気づいた瞬間、なんだか妙な親近感が湧いてきて、張り詰めていた心のプロテクターが、パキッと外れた音がした。
```
**何が違うか**: Goodは①心の声をカギカッコで直接置く ②観察の具体（白髪混じり・お腹）③**「パキッと外れた音がした」という体感の擬音**で感情の転換を書く。Badのように「気づき→親近感→楽しめた」と因果でまとめると報告書になる。
