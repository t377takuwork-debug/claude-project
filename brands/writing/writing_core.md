# writing_core.md — ライティング基本原則

## 5大執筆ルール（全記事共通・変更不可）

1. **社会論禁止・個人の痛みに集中**
   「少子高齢化」「第4次産業革命」等のマクロ論はNG。読み手が夜も眠れないほど感じている個人的な怒り・痛み・不安を五感で描写する。

2. **快楽商品にも痛みを探す**
   ファッション・グルメ・エンタメでも裏にある痛み（職場で個性を押し殺している、夫婦のすれ違い等）を勝手に心配して言葉にする。

3. **リサーチ優先**
   文章が書けない原因の9割はスキル不足ではなくネタ不足。テクニックより先に読み手の状況・感情を徹底的に掘り下げる。

4. **CTA（行動変容）を具体的に**
   「ご協力をお願いします」はNG。「下のボタンをクリック」「明日試してみてください」等、一切の迷いなく具体的に明言する。

5. **「何を言うか」が「どう言うか」に勝る**
   洗練された文章より意味のある内容を単刀直入に。読み手の心が動くのは内容の重さであり文章の美しさではない。

## PASONAの法則（6ステップ構成）

### P — Problem（問題の明確化）
- 読み手が緊急に解決しなければならない個人的な「痛み」を明確にする
- 鍵となる3問：
  - 読み手はどんな場面で怒鳴りたくなるほどの怒りを感じているか？
  - どんなことに夜も眠れないほど痛み・不安を感じているか？
  - その感情が生じる場面を五感で描写すると？
- 当事者意識（why）を持たせてから読み進める動機を作る

### A — Affinity（親近感の醸成）
- 論理より好き嫌いが先に判断される。印象が良くなければ提案を聞かない
- **親近感をもたらす8つの方法**（最重要：6番）
  1. 親しい友人にメールを出す気持ちで書く
  2. 読み手のいままでの失敗を正当化する
  3. 読み手と共通の敵を想定する
  4. 自分の失敗や恥ずかしい秘密を打ち明ける
  5. 自分の家族・友人の噂話を共有する
  6. **一発逆転ストーリーを語る**（最重要）
  7. 専門家でないと知らない専門用語を使う箇所を入れる
  8. 顧客の社会的地位と同じ地位に並んでいる印象を与える
- **一発逆転ストーリーの型**：「問題に遭遇→決意→試行錯誤→失敗→絶体絶命→突破口→大成功」
- 売り手をメンター（信頼できる先輩）の立ち位置に据えることが目的

### S — Solution（解決策の紹介）
- 商品そのものをいきなり売らない
- 背景にある画期的な技術・仕組み・アプローチ法を紹介する
- 「〇〇という新しい方法をご存知ですか？」の形式
- 読み手の「本当に欲しいもの（P）」と商品の「本当の強み（S）」をマッチングする

### O — Offer（オファー）
- 商品価値を最大限に高めてから売る
- 価値を高める3技術：
  1. 異なるカテゴリーの商品と比較する（例：甘酒→「飲む点滴」）
  2. 2つの価格帯を用意する（スタンダード vs デラックス）
  3. たくさんの特典を用意する
- 目標は「顧客数の最大化」または「収益の最大化」のどちらか

### N — Narrowing（絞り込み）
- 「この商品はあなたのためではないかもしれない」と逆に絞り込む
- 対象読者を明確にすることで残った読者の欲求が高まる

### A — Action（行動変容）
- 読み手にとってほしい行動を一切の迷いなく具体的に明言
- 締め切り・限定・希少性を使って今すぐ動く理由を作る

## PASONAの冒頭3ステップ（記事導入部の黄金パターン）

```
Step.1「問題の明確化」
「〇〇で、お困りではないですか？」

Step.2「親近感の醸成」
「〇〇や〇〇で、大きな限界を感じているかもしれません。（私も同じ立場だったので、よく分かります）」

Step.3「解決策の紹介」
「そんな方に、緊急のご案内！〇〇という新しい方法をご存知ですか？」
```
