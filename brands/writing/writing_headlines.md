# writing_headlines.md — 見出し・タイトル作成原則（ケープルズ）

## 見出し5大基本ルール

1. **「得になる」要素を必ず盛り込む**（最重要）
   相手のほしいものがここにあると見出しで知らせる。これを省くのは最大の失敗。

2. **「新情報」があれば大々的に伝える**
   新製品・新手法・新発見はニュース性として全面に出す。

3. **「好奇心」単体で終わらせない**
   好奇心は「得になる」または「新情報」と組み合わせると強力。単体では効果薄。

4. **暗い面・否定的なイメージを避け、明るい面から書く**
   失敗例の7割は否定的表現。プラス面から入ることで読者の感情を上向きに保つ。

5. **手っ取り早く簡単に手に入れられる方法があると伝える**
   ただし「うますぎる話」にならないよう具体的な数字で信頼性を担保する。

---

## 見出しチェックリスト（出力前に全項目確認）

- [ ] 読者の「得になる」要素があるか
- [ ] 「手に入るもの」が具体的に伝わるか
- [ ] タイプ名・フレームワーク名を含んでいないか（mbticodeの場合）
- [ ] 暗い・否定的なイメージを植え付けていないか
- [ ] 一目で意味がわかるか（考えなければ理解できないものはNG）
- [ ] 競合にない差別化された切り口か
- [ ] 好奇心のみで終わっていないか
- [ ] 具体的な数字が入っているか（または入れられないか検討したか）

---

## 見出し成功の4条件（テスト済み）

| 条件 | 説明 |
|---|---|
| 得になる | 相手のほしいものを提供すると伝える（最強） |
| 新情報 | ニュース性・新発見・新手法 |
| 好奇心 | 読者をコピーへ向かわせるフック（単体は弱い） |
| 手っ取り早さ | 短期間・低労力で達成できると伝える |

## 35の見出し型テンプレート

### 【新情報型：8型】
- **型1「ご紹介」で始める**
  `ご紹介します、〇〇` / `ご紹介、まったく新しい〇〇`
- **型2「発表」で始める**
  `発表します。〇〇` / `発表、〇〇の新製品`
- **型3 発表ニュアンスの言葉**
  `ついに登場、〇〇` / `さようなら、〇〇`
- **型4「新」で始める**
  `新登場、〇〇` / `新手法で〇〇`
- **型5「いま・さあ・ついに」で始める**
  `ついに〇〇で！` / `いまなら〇〇が学べます`
- **型6「とうとう・いよいよ」で始める**
  `とうとう出ました！〇〇` / `いよいよ登場！〇〇`
- **型7 日付・年を入れる**
  `この夏、〇〇する方法` / `〇月〇日スタート、〇〇`
- **型8 ニュースネタ風**
  `突然、〇〇できるように` / `発見――〇〇するすばらしい方法`

### 【価格型：5型】
- **型9 価格をメインに**
  `〇〇が、たったの〇〇円`
- **型10 割引価格をメインに**
  `通常〇〇円のところ、〇〇円でセール`
- **型11 特価品をメインに**
  `〇〇日限り。〇〇、〇〇円`
- **型12 支払いの簡単さ**
  `毎週たった〇円で〇〇が手に入ります`
- **型13 無料提供**
  `無料で〇〇を手に入れる方法` / `〇〇を今すぐ無料で`

### 【得になる型（恋愛・人間関係向け）：8型】
- **型14 約束・保証**
  `私に〇日ください。〇〇に変えてみせます`
- **型15 方法・やり方**
  `どうやって〇〇したか` / `こうして私は〇〇しました`
- **型16 理由・ワケ**
  `なぜ〇〇は〇〇なのか` / `〇〇してしまうワケ`
- **型17 比較・対比**
  `〇〇できる人 vs できない人` / `〇〇する人、しない人`
- **型18 失敗する理由**
  `〇〇が失敗する明確な理由` / `〇〇がうまくいかない人の特徴`
- **型19 数字を使う**
  `〇〇する〇つの方法` / `〇〇が変わる〇つのステップ`
- **型20 問いかけ**
  `あなたは〇〇していませんか？` / `〇〇できますか？`
- **型21 一発逆転**
  `〇〇だった私が、〇〇できるようになった方法`

### 【好奇心型：5型】
- **型22 謎かけ**
  `このうちどれが変でしょうか？` / `なぜ〇〇は〇〇なのか？`
- **型23 秘密の公開**
  `〇〇の秘密` / `門外不出の〇〇をついに公開`
- **型24 意外な事実**
  `〇〇の本当の原因は、〇〇ではなかった`
- **型25 ギャップ・逆説**
  `怠け者が成功した理由` / `〇〇しないほうが、〇〇できる理由`
- **型26 証言型**
  `「〇〇をやめたら、〇〇が変わった」` / `〇〇年間悩んでいた〇〇が解決した`

### 【注意・警告型：4型】
- **型27 やってはいけない**
  `〇〇の人がやってはいけない〇〇` / `〇〇する前に知っておくべきこと`
- **型28 カリギュラ（禁止）**
  `この記事は〇〇の人には読んでほしくない`
- **型29 警告**
  `警告！〇〇している人は今すぐ確認してください`
- **型30 呼びかけ**
  `〇〇の方へ` / `〇〇で悩んでいるあなたへ`

### 【ターゲット絞り込み型：5型】
- **型31 お知らせ型**
  `お知らせ：〇〇したい方へ` / `〇〇を目指す方への新しい〇〇`
- **型32 求む型**
  `求む―〇〇したい方` / `〇〇で〇〇を求めています`
- **型33 特定の人へ**
  `いつも〇〇してしまうあなたへ`
- **型34 状況の絞り込み**
  `〇〇回目の〇〇をしているあなたへ`
- **型35 証言＋ターゲット**
  `「〇〇だった私が〇〇できた」と感じている〇〇の方へ`

## 見出しブラッシュアップの鉄則

1. **25本以上書いてから1本選ぶ**（最高品質の確率が上がる）
2. **ひと晩寝かせる**（消費者目線で翌日読み返す）
3. **2通りに解釈できる表現を必ず排除する**
4. **「わかりにくい」見出しは即削除**（読者が考える時間はない）
5. **具体的な数字を入れる**（「5日」「ひと晩で」「3万円」で信頼性を担保）
