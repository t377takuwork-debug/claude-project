# writing_expressions.md — 表現パターン辞典

記事・投稿生成時、必要なアプローチに応じて以下のパターンから最適な表現を選択し出力に反映する。

---

## <problem_patterns>（問題点・不満の言語化）

- **間違い / よくある間違い**
  - キーワード：大間違い、落とし穴、よくあるミス、典型的な間違い
  - 例：「初デートに着ていく服装の典型的な間違いは？」
  - 例：「誰もが必ず犯す5つの間違い――だから、あなたの〇〇はうまくいかない」

- **間違いだらけの**
  - キーワード：ウソだらけ、〇〇にだまされるな！
  - 例：「間違いだらけの〇〇選び。通う前に知っておくべきこと」

- **壁・限界**
  - キーワード：限界、直面、回避、つまずきポイント、リミット
  - 例：「〇〇に立ち塞がる、成長の壁」

- **幻想・ウソ**
  - キーワード：妄想、勘違い、現実は甘くない、信じてはいけない
  - 例：「『自分なんかで〇〇できない』という幻想」
  - 例：「〇〇の常識11のウソ」

- **悪い習慣・うまくいかない人**
  - キーワード：悪習慣、知らないうちに損する、〇〇が苦手な人
  - 例：「いつも恋愛がうまくいかない人の思考パターン」

- **残念な / トラブル / NG**
  - キーワード：やってはいけない、アウト、ありがちな
  - 例：「〇〇はビシッとキマっているのに、残念な〇〇の人」
  - 例：「『頑張れ』はいつからNGワードになったのか」

- **もう古い / 末路**
  - キーワード：時代遅れ、オワコン、陳腐化、自滅、行く末
  - 例：「〇〇はもう古い」
  - 例：「〇〇の末路」

- **知らないと損をする / 知っておくべき**
  - キーワード：知らないではすまされない、マズイぞというニュアンス
  - 例：「知らないと損する！〇〇の『裏ワザと落とし穴』」
  - 例：「知らないと恥をかく〇〇のマナー」

- **罠 / 落とし穴**
  - キーワード：トラップ、蟻地獄、思わぬところでハマる
  - 例：「〇〇する人に潜む〇〇の罠」
  - 例：「順調な〇〇に潜む落とし穴」

- **怖い / 欠けている / 緊急**
  - キーワード：放っておくととんでもないことに、切迫感
  - 例：「本当はこわい〇〇。ただの〇〇と放っておくと…」
  - 例：「〇〇に欠けている視点」

## <desire_patterns>（手に入れたい欲求・失いたくない欲求の刺激）

- **モテる / 稼ぐ / 儲かる**
  - キーワード：生物的欲求、受動的・自動的なニュアンス
  - 例：「脱サラせず『副業で月10万円以上稼ぐ』コツ」
  - 例：「持っているだけで実は稼げる〇〇」

- **勝つ / 負けない / 守る**
  - キーワード：生き残る、損をしない、防衛・安全欲求
  - 例：「〇〇で負けないための〇〇の読み方・使い方」

- **備える / 対策 / 準備**
  - キーワード：安全欲求、予防、時間的な「前」以上の意味
  - 例：「〇〇対策は、早ければ早いほど有利な理由」

- **買いたい / 失敗しない / 後悔しない**
  - キーワード：感情の高まり、機会損失の回避（最強の訴求力）
  - 例：「失敗しない〇〇の極意」
  - 例：「後悔しない〇〇の選び方」

- **脱 / さようなら**
  - キーワード：脱出、問題の側が去っていく
  - 例：「脱・〇〇」
  - 例：「〇〇よ、さらば」

---

## <question_patterns>（問いかけ・プライミング効果）

- **とは？ / なぜ〇〇は〇〇なのか？**
  - キーワード：謎かけ、理由が知りたくなる
  - 例：「なぜ〇〇する人は〇〇できないのか？」

- **どうやって / なぜ一部の人はできるのか**
  - キーワード：方法論、特定の原因の指摘
  - 例：「なぜ一部の人は〇〇で成果を上げられるのか？」

- **こんな〇〇をしていませんか？**
  - キーワード：気づき、痛みを伴う事象に効く
  - 例：「あなたは〇〇でこんな間違いをしていませんか？」
  - 例：「〇〇に、こんな症状が出ていませんか？」

- **もし〇〇だったらどうしますか？**
  - キーワード：自分ごと化、未来の展望予測
  - 例：「もし、あなたの〇〇でこんなことが起こったらどうしますか？」

---

## <curiosity_patterns>（好奇心・意外性の刺激）

- **理由 / ワケ / すべき理由**
  - キーワード：隠された本当の理由、凡庸の回避
  - 例：「〇〇から抜け出せない本当のワケ」
  - 例：「〇〇をやめるべき7つの理由」

- **失敗する理由 / 本当の原因**
  - キーワード：本音と建前、意外性の強調
  - 例：「〇〇が失敗する明確な理由」
  - 例：「〇〇がうまくいかない本当の原因」

- **こうして / いかに**
  - キーワード：物語の予感、ハウツー
  - 例：「こうして私は〇〇した」
  - 例：「平凡だった私がいかにして〇〇になったか」

---

## <gap_patterns>（現実と理想のギャップ・常識の破壊）

- **〇〇が〇〇した（常識と逆の主人公）**
  - 例：「〇〇経験がまったくない私が、〇〇できるようになった」

- **飛躍的な / 前代未聞 / 魔法**
  - 例：「〇〇を飛躍的に伸ばす4つの方法」

- **10倍 / 眠れないほど / 衝撃**
  - 例：「眠れなくなるほどおもしろい〇〇の真実」

---

## <contrast_patterns>（対比による明暗ストーリー）

- **できる vs できない / なれる人・なれない人**
  - 例：「〇〇をつかめる人 vs つかめない人」
  - 例：「年収〇〇万円を稼ぐ人、〇〇万円で止まる人」

- **AではなくB**
  - 例：「〇〇は『〇〇する場』ではなく『〇〇する場』」

- **〇〇よりも効果的**
  - 例：「毎日続けるよりも効果的な〇〇術」

---

## <attention_patterns>（カリギュラ効果・禁止と命令）

- **やってはいけない / 買ってはいけない**
  - キーワード：後悔の回避、デメリットの強調
  - 例：「〇〇している人がやってはいけない3つの習慣」

- **〇〇はもういらない / 〇〇は忘れてください**
  - 例：「ガイドブックはもういらない！〇〇で行くべき〇〇を厳選」

- **警告！ / 注意 / ちょっと待ってください！**
  - 例：「〇〇する人は注意！怖い〇〇」
  - 例：「〇〇に電話？ちょっと待ってください！」
