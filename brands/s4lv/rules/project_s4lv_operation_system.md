## マネタイズ導線

```
Threads（認知・バズ）→ X（信頼構築・2本柱）→ Note無料（週1）→ Note有料（月1）
```

1. **AI実働投稿は有料記事の「生きた証拠」**：「この型は現役で機能している」を日々更新する役割。単体で売り込まない
2. **X→Noteのチラ見せ設計**：有料記事の各章の概念を投稿テーマとして小出しにし、CTAなしで辿り着かせる
3. **無料記事＝有料への装置**：中間CTA×2／テンプレは構造のみ公開・実装は有料／「自力完結の錯覚を壊す一文」をテンプレ直後に
4. **価格戦略**：初弾980〜1,980円（3,000〜5,000字・即効性）で購入実績と感想を貯める→引用できる状態になったら4,980円帯（10,000字超）へ
5. **告知スタンス固定**：「売るのは未熟。だからこそ中身の手順化に徹底的にこだわった」

### 撤退基準（ユーザーのマスターデータ準拠）
- 週20時間以上×1ヶ月継続で目標収益の20%以下 → ピボット
- 3回のアプローチ変更でCV率・エンゲージが微増もしない → ピボット
- 「資産の持続性」が絶対軸。収益の即効性は二次評価

```

### Note無料記事（構成まで）
```
s4lvのNote無料記事に着手。テーマ：[テーマ]。

### Note本文
```

AI臭さ除去の3軸で自己チェックし、qa_article.pyの結果ごと見せて。
```

### 有料記事プロモ
```
[日付]公開の有料Note「[タイトル]」のX予約投稿3本を作って。
6:30予告→12:00公開告知（固定ポスト用・URL入り）→20:00リマインド（収録内容の箇条書き）。
各章の概念チラ見せ設計・本文CTAなしで。
```

### 週次KPI
```

前週比で「伸びた要因1・落ちた要因1・来週変えること1」の3点だけ。
```
