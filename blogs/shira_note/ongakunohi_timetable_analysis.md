## 2. 時間帯別の傾向（3年横断分析）

| 時間帯 | 傾向 |
|---|---|
| **14時台（オープニング）** | 新人・若手グループ（INI、JO1、&TEAM、BE:FIRST、Aぇ! group、超特急、ME:I等）と、郷ひろみ・中西保志・島津亜矢といったベテラン勢が混在してオープニングを飾るのが定番。K-POP勢（ENHYPEN、BOYNEXTDOOR）が入ることも。**郷ひろみは2024・2025と2年連続で14時台に出演しており、2026年ドラフトの「14時台＝郷ひろみ」配置は過去実績と整合する。** |
| **15〜17時台（中盤）** | 坂道グループ（乃木坂46・櫻坂46・日向坂46）、K-POP勢（LE SSERAFIM、NewJeans、TOMORROW X TOGETHER、XG等）がこの帯に集中する傾向。**「希望のうた中継」等の被災地・地方中継企画（2024年：石川・能登＝FUNKY MONKEY BΛBY'S／岩手＝HIPPY／熊本＝WANIMA）が毎年15〜17時台に配置されるのが定番パターン。** 2026年ドラフトが「能登復興祈願中継」を15時台に置き、かつFUNKY MONKEY BΛBY'Sを同時間帯に配置しているのは、2024年の実際の中継企画（FUNKY MONKEY BΛBY'S＠能登）と整合しており根拠がある。 |
| **18〜19時台** | その年のテーマに沿った大型メドレー・合唱企画（2023：GIFTソングカバーメドレー／2024：hope songライブ／2025：ココロ燃やす４大合唱）がこの帯に集中。大合唱企画は19時台が定位置（2023・2024共通）。東京ディズニーリゾートSPパフォーマンスは19〜20時台に配置（年により多少前後）。 |
| **20〜21時台（フィナーレ）** | **事務所の垣根を超えた大型ダンス企画が、3年連続でこの帯の最重要コンテンツになっている**（2023「エンタメ界からダンスのGIFT」9組→2024「垣根は越えた！今度はバトルだ！」18組→2025「DREAM DANCE」95名、と年々規模が拡大）。Snow Manが20〜21時台のクライマックス格ポジションで単独メドレー等を披露するのも2024・2025で共通。 |

## 3. 2026年ドラフトへの示唆（重要）

- **ハーフタイムショーの配置漏れ**：前回のレビューで「ハーフタイムショーがPROJECTSカード・見どころテーブルにはあるが、詳細タイムテーブル本体のどこにも配置されていない」と指摘した点について、2025年実績では「ハーフタイムショー」は**20〜21時台、DREAM DANCE企画と隣接する形で実際に配置されている**ことが確認できた。2026年ドラフトでも20:00台または21:00台のDREAMダンス企画の直前後に配置するのが、過去実績に沿った自然な位置と考えられる。
- **DREAMダンス企画の規模感**：過去3年の同種企画（9組→18組→95名）と比較すると、2026年ドラフトの「7事務所選抜メンバー」という規模感は妥当な延長線上にある。
- **能登中継の時間帯**：2024年の被災地中継企画が15〜17時台に集中していた実績と、2026年ドラフトの「能登復興祈願中継＝15時台」配置は整合している。
