# Mステ 構造化データテンプレート（JSON-LD）

## 更新が必要な箇所（リライトごとに差し替え）

| プレースホルダー | 内容 | 例 |
|---|---|---|
| {M月DD日} | 放送日（表記） | 5月29日 |
| {YYYY-MM-DD} | 放送日（ISO） | 2026-05-29 |
| {dateModified} | 記事更新日時（ISO） | 2026-05-29T10:00:00+09:00 |
| {記事headline} | タイトル | Mステ タイムテーブル【5月29日】... |
| {記事description} | 記事説明文（出演者含む） | - |
| {イベントdescription} | 放送回の説明文 | - |
| {mentions} | 出演者JSON配列 | 下記参照 |
| {performer} | 同上（Event/BroadcastEvent用） | 下記参照 |
| {itemList} | 出演者順番リスト | 下記参照 |
| {FAQ_Q1〜Q4} | FAQのQ&A（記事本文と一致させる） | - |
| {画像URL} | アイキャッチ画像URL | https://shira-treat.com/wp-content/uploads/... |

## 固定箇所（変更不要）

- `datePublished` / `dateCreated`：2025-11-17T18:00:00+09:00
- `author` / `publisher`
- `keywords` / `articleSection` / `inLanguage`
- `about`（番組自体の情報）
- `hasPart`（セクション一覧）
- `BreadcrumbList`
- `location` / `organizer`（テレビ朝日）

## アーティストの型（mentions / performer）

| 区分 | 型 |
|---|---|
| グループ・バンド | `{ "@type": "MusicGroup", "name": "アーティスト名" }` |
| ソロ（個人名） | `{ "@type": "Person", "name": "アーティスト名" }` |
