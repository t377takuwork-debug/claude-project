# Mステアーカイブ記事 JSON-LD テンプレート
# ALL_CAPS の箇所を毎回差し替える。それ以外は固定。
# FAQPage なし（アーカイブ記事のため）

## プレースホルダー一覧

| プレースホルダー | 内容 | 例 |
|---|---|---|
| PAGE_URL | 記事のURL（末尾スラッシュ付き） | https://shira-treat.com/music-station-2026july/ |
| HEADLINE | タイトル（WPタイトル欄と同一） | 【Mステ】2026年7月の出演者や曲目・セトリ一覧（放送アーカイブ） |
| META_DESCRIPTION | メタディスクリプション（同一文字列） | 2026年7月放送「ミュージックステーション」の... |
| DATE_PUBLISHED | 記事初回公開日時（ISO形式） | 2026-07-20T10:00:00+09:00 |
| DATE_MODIFIED | 最終更新日時（ISO形式） | 2026-08-02T10:00:00+09:00 |
| EYECATCH_URL | アイキャッチ画像URL | https://shira-treat.com/wp-content/uploads/... |
| MONTH_JA | 対象月（日本語） | 2026年7月 |
| ARTIST_KEYWORDS | 月内全出演者名（カンマ区切りで列挙） | "SixTONES", "Snow Man", "中森明菜" |
| N_BROADCASTS | 当月の放送回数（整数。Mステは通常月3〜5） | 4 |
| BREADCRUMB_NAME | パンくずリスト最終階層テキスト | 2026年7月アーカイブ |

## 各放送日のListItem ひな形（放送回数分繰り返す）

```
{
  "@type": "ListItem",
  "position": X,
  "item": {
    "@type": "BroadcastEvent",
    "name": "ミュージックステーション（BROADCAST_DATE_JA放送）",
    "startDate": "YYYY-MM-DDT20:00:00+09:00",
    "endDate":   "YYYY-MM-DDT20:54:00+09:00",
    "eventStatus": "https://schema.org/EventScheduled",
    "isLiveBroadcast": true,
    "location": {
      "@type": "Place",
      "name": "テレビ朝日",
      "sameAs": "https://www.tv-asahi.co.jp/"
    },
    "organizer": {
      "@type": "Organization",
      "name": "テレビ朝日",
      "sameAs": "https://www.tv-asahi.co.jp/"
    },
    "performer": [
      // MusicGroup（グループ・バンド）: { "@type": "MusicGroup", "name": "アーティスト名" }
      // Person（ソロ個人名）         : { "@type": "Person",     "name": "アーティスト名" }
      { "@type": "MusicGroup", "name": "ARTIST_NAME_1" },
      { "@type": "Person",     "name": "ARTIST_NAME_2" }
    ]
  }
}
```

> 2時間スペシャルの回は `endDate` を `21:48:00` など実際の放送終了時刻に変更する。
