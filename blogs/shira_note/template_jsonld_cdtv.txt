## プレースホルダー一覧

| プレースホルダー | 内容 | 例 |
|---|---|---|
| HEADLINE | タイトル（WPタイトル欄と同一） | 【CDTV】タイムテーブル 6月8日速報！... |
| META_DESCRIPTION | メタディスクリプション（同一文字列） | CDTVライブ！ライブ！6月8日... |
| DATE_MODIFIED | 更新日時ISO形式 | 2026-06-08T10:00:00+09:00 |
| EYECATCH_URL | アイキャッチ画像URL | https://shira-treat.com/wp-content/... |
| DATE_ISO_TAG | keywords用日付 | 2026-06-08 |
| ARTIST_KEYWORDS | keywordsに追加する出演者名（カンマ区切り） | "SixTONES", "なにわ男子", ... |
| BROADCAST_DATE_JA | 放送日（日本語） | 2026年6月8日 |
| BROADCAST_DESC | BroadcastEvent説明文 | 2026年6月8日（月）19:00〜... |
| START_ISO | 放送開始ISO | 2026-06-08T19:00:00+09:00 |
| END_ISO | 放送終了ISO | 2026-06-08T20:55:00+09:00 ※3時間SPは22:00 |
| N_ARTISTS | 出演者総数（整数） | 8 |
| FAQ_Q1 / FAQ_A1 | Q1の質問文・回答文（本文と完全一致） | |
| FAQ_Q2 / FAQ_A2 | Q2の質問文・回答文（本文と完全一致） | |
| FAQ_Q4 / FAQ_A4 | Q4の質問文・回答文（本文と完全一致） | |
