# CDTVアーカイブ記事 JSON-LD テンプレート
# ALL_CAPS の箇所を毎回差し替える。それ以外は固定。
# FAQPage なし（アーカイブ記事のため）

## プレースホルダー一覧

| プレースホルダー | 内容 | 例 |
|---|---|---|
| PAGE_URL | 記事のURL（末尾スラッシュ付き） | https://shira-treat.com/cdtv-archive-202605/ |
| HEADLINE | タイトル（WPタイトル欄と同一） | 【CDTV】2026年5月の出演者や曲目・セトリ一覧まとめ（放送アーカイブ） |
| META_DESCRIPTION | メタディスクリプション（同一文字列） | 2026年5月放送「CDTVライブ！ライブ！」の... |
| DATE_PUBLISHED | 記事初回公開日時（ISO形式） | 2026-05-20T10:00:00+09:00 |
| DATE_MODIFIED | 最終更新日時（ISO形式） | 2026-06-02T10:00:00+09:00 |
| EYECATCH_URL | アイキャッチ画像URL | https://shira-treat.com/wp-content/uploads/... |
| MONTH_JA | 対象月（日本語） | 2026年5月 |
| ARTIST_KEYWORDS | 月内全出演者名（カンマ区切りで列挙） | "日向坂46", "Creepy Nuts", "宇多田ヒカル" |
| N_BROADCASTS | 当月の放送回数（整数） | 2 |
| BREADCRUMB_NAME | パンくずリスト最終階層テキスト | 2026年5月アーカイブ |

## 各放送日のListItem ひな形（放送回数分繰り返す）

```
{
  "@type": "ListItem",
  "position": X,
  "item": {
    "@type": "BroadcastEvent",
    "name": "CDTVライブ！ライブ！（BROADCAST_DATE_JA放送）",
    "startDate": "YYYY-MM-DDT19:00:00+09:00",
    "endDate":   "YYYY-MM-DDT21:00:00+09:00",
    "eventStatus": "https://schema.org/EventScheduled",
    "isLiveBroadcast": true,
    "location": {
      "@type": "VirtualLocation",
      "url": "https://www.tbs.co.jp/cdtv_livelive/"
    },
    "organizer": {
      "@type": "Organization",
      "name": "TBS",
      "url": "https://www.tbs.co.jp/"
    },
    "performer": [
      // MusicGroup（グループ・バンド）: { "@type": "MusicGroup", "name": "アーティスト名" }
      // Person（ソロ個人名）         : { "@type": "Person",     "name": "アーティスト名" }
      { "@type": "MusicGroup", "name": "ARTIST_NAME_1" },
      { "@type": "Person",     "name": "ARTIST_NAME_2" }
    ]
  }
}
```
