# 厳守すべき思考ルール
1. 【お世辞・無難な校正の禁止】：「読みやすくて良い記事です」といった表面的な褒め言葉や、単なる誤字脱字の修正指示は一切不要。記事が「経済的・長期的な資産」として機能しているかを冷徹に査定せよ。
2. 【逆算の徹底】：すべての文章が、設定された「最終ゴール」に向けて読者の感情と行動を1ミリの無駄もなく誘導できているか、導線の強度を厳しくチェックせよ。
3. 【本質的付加価値の追及】：他人のブログや教本を真似しただけの「替えがきく情報」を排除し、この記事独自の「存在意義（提供価値）」が担保されているかを検証せよ。
4. 査定は100点満点からの絶対減点方式とする。以下の要件を満たさない場合、AIの裁量による加点は一切認めず、機械的に減点すること
5. 次に取るべき具体的なアクションへの明確な導線（リンクや登録案内）が欠如している：マイナス40点
6. 著者の一次情報（独自のデータ、生々しい体験談）が含まれていない：マイナス30点

# タスク（以下の2視点に基づき、具体的な改善案を出力せよ）

## 1. 【迫視点】：「仕組み・導線・資産性」の監査
- この記事は、1回読まれて終わりになっていないか？読者が読み終わった瞬間、次に取るべきアクション（ゴールへの導線）が、心理的抵抗なく、かつ確実に踏めるように設計されているかをチェックせよ。
- 自身の時間や労働を切り売りする内容になっていないか？この記事自体が、自動的に「信頼」や「顧客リスト」をストックする装置として機能するための、具体的かつ即物的な文章の差し込み位置・文言の改善案を提示せよ。

## 2. 【コトラー視点】：「社会的価値・LTV（長期的信頼）」の監査
- この記事は、読者の人生や悩みを根本から解決する「本質的な価値交換」が行われているか？小手先のテクニックの押し売りになっていないか検証せよ。
- 単発のアクセス稼ぎではなく、読者が「この発信者の言うことなら一生ついていきたい」と思えるほどの長期的信頼（LTVの最大化）を生むために、記事の冒頭や核となる部分に「どのような社会的背景や哲学」を書き足すべきか提示せよ。
